# 🚀 Nexe-Agent Generative AI Internship Portfolio

Welcome to my official repository showcasing the complete suite of applications developed during my **Generative AI Internship at Nexe-Agent (Cohort 2026)**. This portfolio demonstrates a progressive learning path from basic conversational interfaces to autonomous multi-tool agents and production-ready RAG pipelines.

---

## 📂 Repository Structure & Project Roadmap

This repository is organized into dedicated folders for each milestone task:

### 🔹 Level 1: Beginner
* **`Task1/` (Conversational AI Chatbot)**
    * **Description:** A secure, interactive terminal-based AI chatbot.
    * **Tech Stack:** Python, Groq API (Llama 3.1).
* **`Task2/` (Secure Email Automation)**
    * **Description:** An automated script that dynamically drafts, formats, and sends secure emails.
    * **Tech Stack:** Python, `smtplib`, `email.message`, Environment Variables.

### 🔹 Level 2: Intermediate
* **`Task3/` (AI Resume Screener)**
    * **Description:** A smart parsing application that extracts data from PDF resumes and scores them contextually against job descriptions.
    * **Tech Stack:** Python, `PyPDF2`, Groq LLM context matching.
* **`Task4/` (WhatsApp FAQ & Auto-Reply Bot)**
    * **Description:** A simulated WhatsApp automation backend running a custom FAQ system, dynamic auto-reply fallback, and conversation logging.
    * **Tech Stack:** Python, Chat Logging File IO.

### 🔹 Level 3: Advanced (Graduation Milestones)
* **`Task5/` (Autonomous Multi-Tool AI Agent)**
    * **Description:** An Advanced Multi-Tool AI Agent that sequentially performs live web scraping, logs structured summaries, and sends automated reports via email.
    * **Tech Stack:** Python, `duckduckgo-search`, `smtplib`, Function execution pipelines.
* **`Task6/` (RAG-based Company Knowledge Assistant)**
    * **Description:** A production-grade Retrieval-Augmented Generation (RAG) system that processes private corporate PDFs/documents into vector space for secure semantic Q&A.
    * **Tech Stack:** Python, `pypdf`, `sentence-transformers` (`all-MiniLM-L6-v2`), `numpy`.
